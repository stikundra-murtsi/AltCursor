# AltCursor

AltCursor - это минималистичный курсор.

Идея AltCursor в том, что в большинстве случаев сам стандартный курсор не заменяется а просто рядом с ним добавляются индикаторы действия.

---
### Установка курсора

#### Windows

Распакуйте архив с курсором, который вы скачали из релизов, и нажмите правой кнопкой мыши на `install.inf` и выберите пункт "Установить". Если курсор не появился сам то выбирете его в ручную в панели управления.

#### X11

Распакуйте архив с курсором, который вы скачали из релизов, и возьмите папку `AltCursor` и перетащите в `/usr/share/icons`, дальше сделайте ссылку: `ln -s /usr/share/icons/AltCursor /usr/share/icons/default`. Дальше установите наш default вот таким образом:

- Откройте в редакторе файл `~/.xinitrc` и введите следующие строки:
```bash
export XCURSOR_THEME=default
export XCURSOR_SIZE=32
```
- Можно так же в ~/.Xresources прописать для стабильности:
```bash
Xcursor.theme: default
Xcursor.size: 32
```

#### LINUX

В основном в DE и в WM главное распаковать архив в /usr/share/icons, а дальше вы можете исходя от вашего DE или WM подключить курсор.
